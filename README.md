# realtime-purchase-streaming
Real-time data pipeline using Google Cloud Pub/Sub and Apache Beam (Dataflow) to stream and transform dummy purchase data into BigQuery. Focuses on cloud-native streaming architecture, reliable pipeline design, and component integration.
